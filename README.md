## 🔍 Project Overview

This project explores the relationship between the **Bitcoin Fear & Greed Index** and the **actual trading performance** of 32 traders on the Hyperliquid decentralised exchange.

**Core questions answered:**
- Do traders make more money during Greed or Fear?
- Which sentiment produces the highest win rate?
- Are LONG or SHORT traders more sentiment-sensitive?
- Which traders perform consistently across all sentiment regimes?
- Is there a statistically significant relationship between sentiment and PnL?

## 📁 Dataset Description

### 1. `historical_data.csv` — Hyperliquid Trader Data
| Column | Description |
|---|---|
| `Account` | Trader wallet address |
| `Coin` | Trading pair (e.g. BTC, ETH) |
| `Execution Price` | Price at trade execution |
| `Size Tokens` | Position size in tokens |
| `Size USD` | Position size in USD |
| `Side` | BUY or SELL |
| `Timestamp IST` | Trade time (Indian Standard Time) |
| `Start Position` | Position before this trade |
| `Direction` | LONG or SHORT |
| `Closed PnL` | Profit/Loss on close (USD) |
| `Fee` | Trading fee paid |
| `Crossed` | Whether order crossed the book |

**Stats:** 211,224 rows · 32 unique traders · 246 coins

### 2. `fear_greed_index.csv` — Bitcoin Fear & Greed Index
| Column | Description |
|---|---|
| `date` | Date (DD-MM-YYYY) |
| `value` | Numeric index score (0–100) |
| `classification` | Extreme Fear / Fear / Neutral / Greed / Extreme Greed |
| `timestamp` | Unix timestamp |

**Stats:** 2,644 rows · Feb 2018 – May 2025

## ⚙️ Setup & Installation

### Prerequisites
- Python 3.9 or higher


### Option A — Run everything at once

## 📈 Key Findings

| Sentiment | Trades | Win Rate | Avg PnL | Total PnL |
|---|---|---|---|---|
| Extreme Fear | 21,400 | 37.1% | $34.54 | $739K |
| Fear | 61,837 | 42.1% | $54.29 | $3.36M |
| Neutral | 37,692 | 39.7% | $35.43 | $1.34M |
| Greed | 50,303 | 38.5% | $42.74 | $2.15M |
| **Extreme Greed** | 39,992 | **46.5%** | **$67.89** | $2.72M |

**Extreme Greed produces the highest win rate (46.5%) and highest average PnL ($67.89).**

## 🧪 Statistical Results

- **Kruskal-Wallis Test:** H = 1225.98, p ≈ 3.7×10⁻²⁶⁴
  → PnL differences across sentiment regimes are **highly statistically significant**
- **Pearson r (sentiment score vs PnL):** r = 0.006, p = 0.006
  → Positive but weak linear correlation — non-linear patterns matter more

## 💡 Insights & Recommendations

1. **Trade more aggressively during Extreme Greed** — win rate peaks at 46.5%, avg PnL at $67.89
2. **Reduce position sizes during Extreme Fear** — worst win rate (37.1%) and lowest avg PnL
3. **Fear regime ≠ bad** — while per-trade quality is lower, high volume means large total PnL opportunity
4. **LONG bias works better in Greed; SHORT in Fear** — see Chart 6 for exact numbers
5. **Sentiment regime shifts (Fear → Greed) signal momentum** — monitor transitions for entries
6. **Top traders are regime-agnostic** — the most consistent accounts profit across all 5 sentiment zones
